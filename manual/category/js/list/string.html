Строки

Основное

        
// создание строки
let str = '';
let str2 = "";

// длина строки
str.length;

// экранирование
str = 'a\b';

// получение символа по индексу
str[0];

// обрезание строки
str.slice(0, 139);

// проверка наличия символа строки
str.indexOf('o'); // если символа в строке нет, вернет -1, в противном случае вернет индекс первого вхождения

Методы объекта

        
// декларация объекта
let obj = {}

// декларация нескольких свйоств у объекта
Object.defineProperty(obj, 'toString', { // toString - это имя свойства
    value: function () {
        return 123;
    }
    enumerable: false; // объект конфигурирования
    configurable: true;
});

Типы свойств:

  • writable - редактируемое свойство или нет;
  • numerable - перечисляемое свойство или нет;
  • configurable - редактирование/удаление свойства (по умолчанию false).

Вернуть данные конфигурирования для объекта и свойства объекта:

        
Object.getOwnPropertyDescriptor(obj,'text'); // obj - объект, 'text' - свойство

Numerable - скрывает свойство в списке свойств при вызове Object.keys.

Геттеры и сеттеры

Это альтернативный метод декларации свойств и методов объекта.

get: обращение к свойству на чтение.

get: обращение к свойству на приравнивание (установить свойство).

Заморозка

        
Object.isFrozen(obj); // заморожен объект или нет
Object.freeze(obj); // заморозить объект (меняет параметры writable и configurable в true)
