Исключения

Выбросить исключение:

        
if (isNaN(likes) || likes < 0) {
    throw new TypeError('Неверное значение');
}

Обработка исключений:

        
try {
    // TODO
} catch (e) {
    if (e instanceof TypeError) {
        // TODO
    }
}

TypeError:

  • e.name - имя исключения
  • e.message - сообщение ошибки
  • e.stack - стек вызовов до исключения

console.error('ошибка') - вывод в консоль ошибки
