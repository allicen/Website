Функции

Функции - это объекты высшего порядка. Всегда возвращают undefined, если нет return.

Передача параметров:

  • по значению (внутри функции создается копия параметра, начальное значение не меняется);
  • по ссылке (все изменения сохраняются, начальное значение меняется).

Область видимости:

  • глобальная;
  • область видимости функции.

Затенение - переопределение переменной функции внутри другой вложенной функции.

Всплытие - обращение к переменной до е объявления. Интерпритатор сначала инициализирует код, и только потом начинает его выполнять.

Замыкание - функция вместе со всеми внешними переменными, которые ей доступны.

Если на переменную никто не ссылается, то сборщик мусора ее вычистит.

Декларация:

        
// наиболее распространенная
function name () {}

// в виде строки
new Function('тело функции в виде строки');

// самовызывающаяся функция
let func = (function () {
    // TODO
} ());

Аргументы

  • arguments - массивоподобный объект с аргументами функции
  • arguments.length - длина массива с аргументами
  • [].slice.call(arguments) - массив аргументов
