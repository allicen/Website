Числа

Префиксы для чисел

        
0x11 === 17 // 16ричная запись
0b11 === 3 // 2ичная запись
0o11 === 9 // 8ричная запись

2.998e8 === 2.998 * 10^8 // экспонентная запись числа

Приведение к числу

        
// parseInt и parseFloat возвращает число, которое смог прочитать (число до первого нечислового символа)
parseInt('123', 10); // любые систеым счисления
parseFloat('123.4'); // только 10ричная система счисления

// Number возвращает число, если вся переданная строка - число
Number('10');
