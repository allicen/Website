Массивы

1. Создание массива:

        
let arr = []

2. Длина массива:

        
arr.length

3. Обход массива:

        
// с помощью for
for (let i = 0; i < arr.length; i++) {
    if (arr[i] > 20) {
        result.push(item);
    }
}

// с поомщью forEach
arr.forEach(f = (item) => {
    if (item > 20) {
        result.push(item);
    }
});

// с помощью filter
let result = arr.filter(f = (item) => { return item > 20; });

// с помощью reduce
// аккумулятором может быть любой тип данных
let arr = tweets.reduce(hashtags, []); // hashtags - функция обратного вызова, [] - аккумулятор и начальное значение
function hashtags (acc, item) {
    return arr.concat(item.hashtags || []);
}

4. Добавление элемента в конец массива:

        
arr.push(elem);

5. Выталкивание элемента из массива. Удаление последнео элемента массива:

        
arr.pop();

6. Объединение массивов:

        
arr.concat(arr2);

Начальные массивы остаются неизменными.

7. Срез массива. Получение части массива:

        
arr.slice(0, 5); // 0 - начальная позиция среза, 5 - конечный элемент среза
arr.slice(); // получение копии массива

8. Удаление и замена элементов массива:

        
arr.splice(4, 1, 'строка'); // 4 - с какого элемента, 1 - количество элементоа для удаления, 'строка' - добавить в массив

9. Функции для массивов:

        
arr.sort() // сортировка массива в лексикографическом порядке
arr.every() // удовлетворяют ли все элементы массива условию
arr.some() // удовлетворяет ли хотя бы один элемент условию
arr.shift() // выталкивает из массива первый элемент и возвращает его
arr.unshift() // добавляет элемент в начало массива

Пример:

[10, 20, 30].every(elem => elem >= 10);
