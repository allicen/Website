Операторы

Оператор - это метод.

Инфиксное определение операторов.

        
val sum = 1 + 2.+(3)
print(sum) // 6

val mul = 3.*(2) * 2
print(mul) // 12

Перегразка операторов:

        
case class Vector2D(x: Double, y: Double) {
    // скалярное произведение
    def * (vec2D: Vector2D) = x * vec2D.x + y * vec2D.y
}

val e = Vector2D(1.0, 1.0)
val mul = Vector2D(-2.0, 3.0) * e
print(mul) // 1.0
