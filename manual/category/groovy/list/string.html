Получение символа строки или подстроки по индексу

        
class Example {
   static void main(String[] args) {
      String sample = "Hello world"

      // Получить символ по индексу
      println(sample[4]) // o

      // Получение последнего символа
      println(sample[-1]) // d

      // Получение подстроки
      println(sample[1..2]) // el - с 1 по 2 символ слева направо
      println(sample[4..2]);// oll - с 4 по 2 символ справа налево
   }
}

Присутствие символа в строке indexOf( )

Возвращает индекс первого вхождения символа/подстроки. Если ничего не найдено, выводит -1.

Синтексис:

        
public int indexOf(int ch) // Поиск символа
public int indexOf(int ch, int fromIndex) // fromIndex - поиск с индекса
int indexOf(String str) // Поиск подстроки
int indexOf(String str, int fromIndex) // fromIndex - поиск с индекса

Примеры:

        
String a = "Hello World"

println(a.indexOf('e')) // 1
println(a.indexOf('o')) // 4

println(a.indexOf('l',1)) // 2
println(a.indexOf('e',4)) // -1

println(a.indexOf('el')) // 1
println(a.indexOf('or')) // 7

println(a.indexOf('el',1)) // 1
println(a.indexOf('or',8)) // -1

Заполнение строки символами слева и справа до указанной длины padLeft( ), padRight( ), center( )

Синтаксис:

        
String padLeft(Number numberOfCharacters) // Дополнитбь строку пробелами слева
String padLeft(Number numberOfCharacters, String padding) // Дополнить строку указанными символами слева

String padRight(Number numberOfCharacters) // Дополнитбь строку пробелами справа
String padRight(Number numberOfCharacters, String padding) // Дополнить строку указанными символами справа

String center(Number numberOfChars) // Расположение строки по центру, заполнение пробелами слева и справа

Примеры:

        
String a = "Hello World"

println(a.padRight(16)) // Hello World
println(a.padRight(16,'*')) // Hello World*****

println(a.padLeft(16)) //         Hello World
println(a.padLeft(16,'*')) // *****Hello World

Сравнение строк без учета регистра equalsIgnoreCase( ), compareToIgnoreCase( )

        
// Получение true/false

String a = "Hello World"
String b = "HELLO World"
String c = "HELLO WORLD"

a.equalsIgnoreCase(b) // true
a.equalsIgnoreCase(c) // true
b.equalsIgnoreCase(c) // true

// Получение информации о том, какая строка длиннее и насколько

String str1 = "Hello World"
String str2 = "HELLO WORLD"
String str3 = "HELLO World World"

str1.compareToIgnoreCase( str2 ) // 0 - строки равны
str2.compareToIgnoreCase( str3 ) // -6 - str2 короче str3 на 6 символов
str3.compareToIgnoreCase( str1 ) // 6 - str3 длиннее str1 на 6 символов
