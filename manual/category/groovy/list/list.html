Добавление в список add( )

1. Синтаксис:

        
boolean add(Object value)

Параметры:

  • value - Значение, которое должно быть добавлено в список.

Возвращаемое значение: true/false в завимости от того, был ли добавлен элемент.

2. Синтаксис:

        
void add(int index, Object value)

Параметры:

  • value - Значение, которое должно быть добавлено в список.
  • index - Индекс, с которого нужно добавить элементы.

Пример:

        
def lst = [11, 12, 13, 14]
lst.add(15) // [11, 12, 13, 14, 15]
lst.add(2,20) //[11, 12, 20, 13, 14, 15]

Содержит ли список определенное значение contains( )

Синтаксис:

        
boolean contains(Object value)

Параметры:

  • value - Значение, которое нужно найти в списке.

Возвращаемое значение: true/false в завимости от того, был ли найден элемент.

Пример:

        
def lst = [11, 12, 13, 14];
lst.contains(12); // true
lst.contains(18); // false

Получить элемент get( )

Синтаксис:

        
Object get(int index)

Параметры:

  • index - Индекс элемента.

Возвращаемое значение: объект в указанной позиции.

Пример:

        
def lst = [11, 12, 13, 14]
lst.get(0) // 11
lst.get(2) // 13

Проверка списка на пустоту isEmpty( )

Синтаксис:

        
boolean isEmpty()

Возвращаемое значение: true - если список пустой, false - если не пустой.

Пример:

        
def lst = [11, 12, 13, 14]
def emptylst = []

lst.isEmpty() // false
emptylst.isEmpty() // true

Разность списков minus( )

Метод создает новый список, в котором не будет указанных элементов.

Синтаксис:

        
List minus(Collection collection)

Параметры:

  • collection - Список со значениями.

Возвращаемое значение: новый список.

Пример:

        
def lst = [11, 12, 13, 14]
def newlst = []

newlst = lst.minus([12,13]) // [11, 14]

Объединение списков plus( )

Метод создает новый список, в котором будут значения исходных списков.

Синтаксис:

        
List plus(Collection collection)

Параметры:

  • collection - Список со значениями.

Возвращаемое значение: новый список.

Пример:

        
def lst = [11, 12, 13, 14]
def newlst = []

newlst = lst.plus([15,16]) // [11, 12, 13, 14, 15, 16]

Удаление последнего элемента из списка pop( )

Синтаксис:

        
Object pop()

Возвращаемое значение: удаленный из списка элемент.

Пример:

        
def lst = [11, 12, 13, 14]

lst.pop() // 14
println(lst) // [11, 12, 13]

Удаление элемента по индексу remove( )

Синтаксис:

        
Object remove(int index)

Параметры:

  • index - Позиция элемента.

Возвращаемое значение: удаленный элемент.

Пример:

        
def lst = [11, 12, 13, 14]

lst.remove(2) // 13
println(lst) // [11, 12, 14]

Разворот списка reverse( )

Метод создает новый список, в котором все элементы исходного списка располагаются в обратном порядке.

Синтаксис:

        
List reverse()

Возвращаемое значение: новый список.

Пример:

        
def lst = [11, 12, 13, 14]
def revlst = lst.reverse() // [14, 13, 12, 11]

int size()

def lst = [11, 12, 13, 14]
