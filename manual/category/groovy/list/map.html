Добавление значений

        
def sampleMap = [:]

sampleMap.put('thickness', 10)
sampleMap['color'] = 'Blue'
sampleMap.weight = 500
sampleMap.'shape' = 'Circle'
sampleMap << [price:150]

Замена значений

        
def sampleMap = [color:'Blue', weight:500, shape:'Circle', thickness:10, price:150]

sampleMap['color'] = 'Red'
sampleMap.'shape' = 'Rectangle'
sampleMap.weight = 200
sampleMap.put('thickness', 5)
sampleMap << [price:99]

Удаление значений

        
def sampleMap = [color:'Blue', weight:500, shape:'Circle', thickness:10, price:150]

sampleMap.remove('color')
sampleMap = sampleMap - ['shape':'Circle']
sampleMap -= [weight:500]

Получение значений

        
def sampleMap = [color:'Blue', weight:500, shape:'Circle', thickness:10, price:150]

println "The color is: ${sampleMap.get('color')}"
println "The weight is: ${sampleMap['weight']}"
println "The shape is: ${sampleMap.shape}"
println "The thickness is: ${sampleMap.'thickness'}"

Содержит ли эта карта этот ключ containsKey( )

        
class Example {
   static void main(String[] args) {
      def mp = ["TopicName" : "Maps", "TopicDescription" : "Methods in Maps"]

      mp.containsKey("TopicName") // true
      mp.containsKey("Topic") // false
   }
}

Найти значение по ключу get( )

Если ключа нет, возвращает null.

        
class Example {
   static void main(String[] args) {
      def mp = ["TopicName" : "Maps", "TopicDescription" : "Methods in Maps"]

      mp.get("TopicName") // Maps
      mp.get("Topic")  // null
   }
}

Набор ключей карты keySet( )

        
class Example {
   static void main(String[] args) {
      def mp = ["TopicName" : "Maps", "TopicDescription" : "Methods in Maps"]
      mp.keySet()  //  [TopicName, TopicDescription]
   }
}

Добавление и замена значений put( )

Связывает указанное значение с указанным ключом на этой карте. Если эта карта ранее содержала сопоставление для этого ключа, старое значение заменяется указанным значением.

        
class Example {
   static void main(String[] args) {
      def mp = ["TopicName" : "Maps", "TopicDescription" : "Methods in Maps"]
      mp.put("TopicID","1")
      println(mp) // [TopicName:Maps, TopicDescription:Methods in Maps, TopicID:1]
   }
}

Размер карты size( )

Возвращает количество элементов в карте.

        
class Example {
   static void main(String[] args) {
      def mp = ["TopicName" : "Maps", "TopicDescription" : "Methods in Maps"]
      mp.size() // 2

      mp.put("TopicID","1")
      mp.size() // 3
   }
}

Значения карты values( )

        
class Example {
   static void main(String[] args) {
      def mp = ["TopicName" : "Maps", "TopicDescription" : "Methods in Maps"]
      mp.values() // [Maps, Methods in Maps]
   }
}
