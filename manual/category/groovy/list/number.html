Получение значения числа другого типа *Value( )

        
class Example {
   static void main(String[] args) {
      Integer x = 5

      // Конвертирует в double
      println(x.doubleValue()) // 5.0

      // Конвертирует в byte
      println(x.byteValue()) // 5

      // Конвертирует в float
      println(x.floatValue()) 5.0

      // Конвертирует в long
      println(x.longValue()) // 5

      // Конвертирует в short
      println(x.shortValue()) // 5

      // Конвертирует в int
      println(x.intValue()) // 5
   }
}

Сравнение чисел compareTo( )

        
class Example {
   static void main(String[] args) {
      Integer x = 5

      // Первое число больше второго
      x.compareTo(3) // 1

      // Числа равны
      x.compareTo(5) // 0

      // Первое число меньще второго
      x.compareTo(8) // -1
   }
}

class Example {

Получить объект числа с указанным значением valueOf( )

Синтаксис:

        
static Integer valueOf(int i)
static Integer valueOf(String s)
static Integer valueOf(String s, int radix) // radix - основание системы счисления

Пример:

        
class Example {
   static void main(String[] args) {
      int x = 5
      Double z = 15.56

      Integer xNew = Integer.valueOf(x)
      println(xNew) // 5

      Double zNew = Double.valueOf(z)
      println(zNew) // 15.56
   }
}

Округление чисел ceil( ), floor( ), rint( ), round( )

Возвращение double-значений:
