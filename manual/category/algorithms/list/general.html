Сумма цифр числа (Java):

        
/**
 * @param number Число
 * @return Сумма цифр числа
 * */
private static int sumNumberSymbol(int number) {
    int sum = 0;
    while (number != 0) {
        sum += (number % 10);
        number /= 10;
    }

    return sum;
}

Проверка числа на простоту (Java):

        
/**
 * @param number Число
 * @return true если число простое, false - иначе
 * */
private static boolean isSimple(int number) {
    double sqrt = Math.sqrt(number);

    if(number == 1 || number == 2) return true;
    if(number % 2 == 0) return false;

    for(int i = 3; i <= sqrt; i += 2){
        if(number % i == 0) return false;
    }
    return true;
}

Список простых множителей числ (Java):

        
/**
 * @param number Число
 * @return Список простых множителей числа
 * */
private static ArrayList<Integer> simpleMultipliers(int number) {
    ArrayList<Integer> multipliers = new ArrayList<>();
    for (int i = 2; i <= number;) {
        if (isSimple(i) && number % i == 0) {
            multipliers.add(i);
            number /= i;
        } else {
            i++;
        }
    }

    return multipliers;
}

Является ли год високосным (Java):

        
/**
 * @param number Число
 * @return Високосный год или нет
 * */
private static boolean leap(int year) {
    return (year % 4 == 0 && year % 100 != 0) || year % 400 == 0;
}

Факториал числа (Java):

        
/**
 * @param number Число
 * @return Факториал числа
 * */
private static int fact(int number) {
    int count = 1;
    for (int i = number; i > 1; i--) {
        count *= i;
    }
    return count;
}

/**
 * @param numberInput Число
 * @param numberSystem Система счисления
 * @return Число в заданной системе счисления
 * */
private static int nok (int a, int b) {
    int c = a, d = b;
    while (a != b){ // Используется алгоритм Евклида
        if(a > b) a = a - b;
        else b = b - a;
    }
    return c / a * d;
}
