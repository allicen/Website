За сколько ходов слон доберется до клетки:

        
private static int stepCount() {
    if (Math.abs(x2-x1) == Math.abs(y2-y1)) {
        return 1;
    } else if ((x1+y1) % 2 == (x2+y2) % 2) {
        return 2;
    }
    else  {
        return 0;
    }
}
